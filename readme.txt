Содержание симулятора для идеально симметричного случая с сферическими 
волновыми фронтами:
Simulator_spherical_WF.m - главный файл симулятора
deriv.m - функция, которая считает производные сферического ВФ 
          для заданного расстояние между источником и ДШГ L_def
Hartmannograms - папка, в которую идет запись гартманнограм 
                 (должна быть создана до записи)
Files_txt - папка, в которую идет запись текстовых (.txt) файлов 
            для обработки (должна быть создана до записи)

Данная версия симулятора выдает корректные результаты только для
симметричного случая с размером питча 137,5 мкм и размером пикселя 
5,5 мкм. Могут быть сформированы только ВФ с дефокусировкой.
Входными данными для симулятора служат параметры ДШГ: размер питча,
пикселя, фокусное расстояние растра, длина волны принимаего излучения,
кол-во пикселей сенсора по осям x и y, битность камеры.
Выходные данные симулятора (гартманнограма плоского референса, текстовый
референс, обрабатываемая гартманнограма и обрабатываемый текстовый файл)
получены по заданному расстоянию.между источником и ДШГ L_def
Идея: 
1) Программа в каждой микролинзе формирует матрицу интенсивности, 
соответсвующую дифракционной картине на квадратном отверстии:
I(x,y) = (sinc(Pitch*x/Lambda/ML_focal))^2 * (sinc(Pitch*y/Lambda/ML_focal))^2
2) Далее интегрирует по пикселям и конвертирует это представление в 
цифровое (квантование по уровням битности).
3) По заданному радиусу ВФ (то же L_def) с помощью функции deriv.m
рассчитывают производные cathbxtcrjuj ВФ по осям х и у. 
Смещение центра фокального пятна вычисляется как:
Shift = WF_deriv * ML_focal
4) Тогда матрица интенсивности рассчитывается след образом:
I(x,y) = (sinc(Pitch*(x-ShiftX)/Lambda/ML_focal))^2 * (sinc(Pitch*(y-ShiftY)/Lambda/ML_focal))^2
